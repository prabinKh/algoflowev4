Usage

Run the master test runner which will execute every `test_*.py` script in this
folder and print a per-app summary:

```bash
python command/run_all_tests.py
```

You can also run individual app scripts:

```bash
python command/test_eadmin.py
python command/test_efrontend.py
python command/test_chatapp.py
python command/test_account.py
```
